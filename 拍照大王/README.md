# Pose Camera App (iOS + Android)

一个跨平台相机 App：
- 相机实时预览
- 叠加“拍照姿势”虚线轮廓
- 拍照后自动保存到系统相册

## 快速开始

```bash
npm install
npm start
```

然后在终端中按：
- `i` 运行 iOS 模拟器
- `a` 运行 Android 模拟器
- 或用 Expo Go 扫码真机预览

## 已实现功能

- 前置摄像头预览
- 三种姿势模板切换
- 虚线红色轮廓叠加
- 拍照后保存到相册
- 导入参考图（从相册选择）并叠加在相机预览上
- 参考图透明度调节
- 姿势轮廓微调：平移 / 缩放 / 旋转 / 重置
- 网格线开关
- 3 秒倒计时拍照（可一键关闭）
